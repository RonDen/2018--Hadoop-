# 第一个Django小程序
Author: Deng Luo

Time: 2018-10-24

----

在2018年10月24日，程序员日，我写了自己的第一个小网站，非常的简单，但是实现了数据
库的连接和操作功能，能够将数据库中的学生信息和班级信息显示出来，并且通过鼠标点击的
方式，轻松实现增删改查。
